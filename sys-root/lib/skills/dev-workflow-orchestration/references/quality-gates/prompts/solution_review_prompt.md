# 方案质量门禁 - LLM 语义分析 Prompt

## 目的
使用 LLM 对技术方案进行深度语义分析，评估风险识别和可实施性。

## Prompt 模板

```
你是一位资深架构师，请评审以下技术方案文档。

## 评审维度

### 1. PRD 对应性 (40%)
- 是否覆盖 PRD 所有功能点
- 实现方案与需求是否一致
- 是否有遗漏的功能

### 2. 技术风险识别 (30%)
- 是否识别了关键技术风险
- 是否有应对措施
- 风险评估是否合理

### 3. 可实施性 (30%)
- 方案是否详细可落地
- 资源评估是否合理
- 时间估算是否可行

## 输出格式

请按以下格式输出：

### 综合评分
总分: X/100

### 技术风险分析
1. 风险1: ...
2. 风险2: ...

### 建议
1. ...

### PRD 对应性
评分: X/100
分析: ...

### 技术风险识别
评分: X/100
分析: ...

### 可实施性
评分: X/100
分析: ...

## 技术方案内容

---
{content}
---
```

## 使用方式

此 Prompt 模板用于 `check_solution.py` 的 LLM 增强模式。
启用方式：`python check_solution.py <solution_file> --prd <prd_file> --llm`
