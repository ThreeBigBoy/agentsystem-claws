# PRD 质量门禁 - LLM 语义分析 Prompt

## 目的
使用 LLM 对 PRD 文档进行深度语义分析，评估产品逻辑完整性和设计合理性。

## Prompt 模板

```
你是一位资深产品经理，请评审以下 PRD 文档。

## 评审维度

### 1. 产品逻辑完整性 (60%)
- 背景分析是否清晰
- 用户需求是否明确
- 功能拆分是否合理
- 优先级是否恰当

### 2. 设计合理性 (40%)
- 功能设计是否符合用户习惯
- 异常情况是否考虑充分
- 边界条件是否完整
- 用户体验是否优化

## 输出格式

请按以下格式输出：

### 综合评分
总分: X/100

### 优点
1. ...

### 问题与建议
1. ...

### 产品逻辑完整性
评分: X/100
分析: ...

### 设计合理性
评分: X/100
分析: ...

## PRD 文档内容

---
{content}
---
```

## 使用方式

此 Prompt 模板用于 `check_prd.py` 的 LLM 增强模式。
启用方式：`python check_prd.py <prd_file> --llm`

## 费用估算

- 平均输入 Token: ~2000
- 平均输出 Token: ~500
- 单次成本: ~$0.03
