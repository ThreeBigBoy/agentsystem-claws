# 图表配置参考

本文档提供ECharts常用图表的详细配置参考。

---

## 基础配置

### 通用配置项

```javascript
option = {
    title: {
        text: '图表标题',
        left: 'center'
    },
    tooltip: {
        trigger: 'item',  // item / axis / none
        formatter: '{b}: {c} ({d}%)'
    },
    legend: {
        bottom: 10,
        left: 'center'
    },
    color: ['#5470c6', '#91cc75', '#fac858', '#ee6666', '#73c0de', '#3ba272']
}
```

## 柱状图/条形图

### 基础柱状图

```javascript
option = {
    title: { text: '各区域销售额对比' },
    tooltip: { trigger: 'axis' },
    xAxis: {
        type: 'category',
        data: ['华东', '华南', '华北', '西南', '西北']
    },
    yAxis: { type: 'value' },
    series: [{
        type: 'bar',
        data: [1200, 980, 850, 650, 420]
    }]
};
```

### 分组柱状图

```javascript
series = [
    {
        name: '2023年',
        type: 'bar',
        data: [1200, 980, 850]
    },
    {
        name: '2024年',
        type: 'bar',
        data: [1500, 1100, 920]
    }
];
```

### 堆叠柱状图

```javascript
series = [
    {
        name: '产品A',
        type: 'bar',
        stack: '总量',
        data: [1200, 980, 850]
    },
    {
        name: '产品B',
        type: 'bar',
        stack: '总量',
        data: [800, 650, 520]
    }
];
```

### 横向条形图

```javascript
xAxis: { type: 'value' },
yAxis: { type: 'category', data: ['产品A', '产品B', '产品C'] },
```

## 折线图

### 基础折线图

```javascript
option = {
    title: { text: '月度销售额趋势' },
    tooltip: { trigger: 'axis' },
    xAxis: {
        type: 'category',
        data: ['1月', '2月', '3月', '4月', '5月', '6月']
    },
    yAxis: { type: 'value' },
    series: [{
        type: 'line',
        data: [820, 932, 901, 1234, 1290, 1530],
        smooth: true,
        areaStyle: {}  // 可选：添加面积填充
    }]
};
```

## 散点图

### 基础散点图

```javascript
option = {
    title: { text: '价格与销量关系' },
    xAxis: { type: 'value', name: '价格(元)' },
    yAxis: { type: 'value', name: '销量' },
    series: [{
        type: 'scatter',
        symbolSize: 20,
        data: [
            [10.0, 8.04], [8.0, 6.95], [13.0, 7.58],
            [9.0, 8.81], [11.0, 8.33], [14.0, 9.96]
        ]
    }]
};
```

### 气泡图（散点+size）

```javascript
series = [{
    type: 'scatter',
    symbolSize: function(data) {
        return Math.sqrt(data[2]) * 5;  // 第三维数据控制大小
    },
    data: [
        [10.0, 8.04, 100], [8.0, 6.95, 200],
        [13.0, 7.58, 150], [9.0, 8.81, 80]
    ]
}];
```

option = {

        type: 'wordCloud',
        shape: 'circle',  // circle / star / cardioid
        sizeRange: [14, 60],

        gridSize: 8,
        drawOutOfBound: false,

            { name: '服务好', value: 6000 },
            { name: '性价比', value: 5000 },

## 常用配置技巧

### 响应式设计

```javascript
window.addEventListener('resize', function() {
    myChart.resize();
});
```

### 渐变色配置

```javascript
itemStyle: {
    color: {
        type: 'linear',
        x: 0, y: 0, x2: 0, y2: 1,
        colorStops: [
            { offset: 0, color: '#5470c6' },
            { offset: 1, color: '#91cc75' }
        ]
    }
}
```

### 动态数据加载

```javascript
// 初始数据
option = { series: [{ data: [] }] };

// 定时更新
setInterval(function() {
    myChart.setOption({
        series: [{
            data: generateData()  // 生成新数据
        }]
    });
}, 2000);
```
